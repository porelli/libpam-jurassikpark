libpam-jurassikpark
===================

PAM module that show a video like in Jurassik Park film after three failed logins

Building
--------

```$ cd src
$ tar cvzf ../libpam-jurassikpark_0.1.orig.tar.gz usr Makefile ; debuild -us -uc```

Installing
----------

```# dpkg -i ../libpam-jurassikpark*.deb```
